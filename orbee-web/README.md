# 🐝 Orbee | TaskFlow Kanban

> **Status:** Em desenvolvimento Ativo 🚀

![Dashboard Orbee](./assets/dashboard.png)

O **Orbee** é um ecossistema inteligente de gestão de tarefas projetado para desenvolvedores que buscam alta performance sem abrir mão de uma interface moderna e fluida. O projeto utiliza o conceito de **Glassmorphism** e **Clean Architecture** para entregar uma experiência de usuário (UX) superior.

---

## 🛠️ Stack Tecnológica

### **Frontend**
* **React + Vite:** Performance e HMR instantâneo.
* **TypeScript:** Segurança de tipos e escalabilidade.
* **Tailwind CSS:** Estilização utilitária com foco em responsividade.
* **Lucide React:** Iconografia moderna e leve.

### **Backend (Em progresso)**
* **Python + FastAPI:** Backend de alta performance com validação via Pydantic v2.
* **SQLite:** Banco de dados relacional leve para persistência local.

---

## 🏗️ Arquitetura e Diferenciais

* **Clean Architecture:** Separação clara de responsabilidades entre UI, Tipagem e Lógica de Negócio.
* **Interface Flutuante:** Sidebar adaptativa que maximiza o espaço de trabalho.
* **Navegação Dinâmica:** Troca inteligente entre as visualizações **Diária, Semanal e Mensal**.
* **Visual Neon/Glass:** Estética moderna inspirada nos melhores setups de desenvolvimento.
